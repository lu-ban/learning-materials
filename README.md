## algorithms

算法方面的学习通过三个阶段来不断加强这方面的能力:
- 首先通过阅读与练习《算法图解》来对算法有一个基本的了解，阅读完之后对本书中的知识点进行总结与练习
- 第二步通过《编程珠玑》来加强对算法的理解度、扩大知识面
- 最后通过《算法导论》来系统、细致的再深入学习算法的原理、应用等
